
# 📊 Site Agenda — Consultoria de Crédito Planejado

Landing page desenvolvida para captação de clientes interessados em **crédito planejado (imóvel e veículo)** com simulação simples e envio direto via WhatsApp.

---

## 🚀 Sobre o projeto

O **Site Agenda** é uma página web responsiva criada para:

- Simular crédito de forma rápida
- Facilitar o contato direto com o cliente via WhatsApp
- Apresentar o serviço de forma clara e profissional
- Gerar leads qualificados

---

## 🧩 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- GitHub Pages (Hospedagem)

---

## 📱 Funcionalidades

- ✅ Simulação de crédito (imóvel e veículo)
- ✅ Cálculo automático de parcelas
- ✅ Envio de dados direto para WhatsApp
- ✅ Interface responsiva (mobile e desktop)
- ✅ Depoimentos de clientes
- ✅ Vídeos explicativos (YouTube)
- ✅ Layout moderno e profissional
